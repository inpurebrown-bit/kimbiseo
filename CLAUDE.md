# 🤖 김비서 (KimBiseo) - 업무 관리 대시보드

**스마트 업무 브리핑 및 회의 자료 생성 시스템**

---

## 📋 프로젝트 개요

**김비서**는 팀의 일일 업무, 프로젝트 진행 상황, 매출 현황을 자동으로 수집하고 분석하여 회의용 브리핑 자료를 생성하는 통합 업무 관리 대시보드입니다.

### 🎯 주요 목표
- ✅ 업무 데이터의 중앙화된 관리
- ✅ 회의용 자료 자동 생성
- ✅ 실시간 프로젝트 진행 현황 추적
- ✅ 매출 데이터 시각화 및 분석
- ✅ 팀 일정 및 업무 우선순위 자동 정렬

---

## 🎨 주요 기능

### 1. **📊 통합 대시보드**
- **Glassmorphism UI** 디자인으로 현대적 외관
- **다크/라이트 모드** 자동 전환 (localStorage 저장)
- **2×2 그리드 레이아웃**으로 4개 주요 섹션 표시
  - 할 일 목록 (오늘/이번주)
  - 주간 일정
  - 프로젝트 진행률
  - 매출 요약

### 2. **📈 매출 분석**
- Canvas 기반 **월별 매출 추이** 선 그래프
- **제품별 비교** 막대 그래프
- 자동 숫자 포맷팅 (한국 콤마)
- KPI 카드 (생산성, 품질, 일정준수, 비용효율)

### 3. **📊 프로젝트 관리**
- SVG 기반 **5단계 프로세스 플로우** (기획→제작→검토→배포→분석)
- 진행률 바 시각화
- 예산 집행 현황
- 리소스 할당 파이 차트
- KPI 섹션

### 4. **💰 평균 가격 분석**
- 전체 거래 평균 단가 계산 (₩51,500)
- 제품별 평균 이상/이하 비교
- 가격대별 분류 및 분석
- 30건 모든 거래 데이터 테이블

### 5. **📋 데이터 관리**
- CSV 파일 기반 영구 저장소
- TXT 파일로 텍스트 데이터 관리
- 자동 파일 읽기 및 파싱

### 6. **🤖 김비서 AI 어시스턴트**
- `/김비서` 슬래시 명령어
- 모든 데이터 자동 수집
- 회의용 브리핑 자동 생성
- 오늘/내일/이번주 업무 자동 분류
- 우선순위 기반 정렬
- 마크다운 형식 출력

## 🛠️ 사용 기술

### 프론트엔드
- **HTML5** - 시맨틱 마크업
- **CSS3** - Glassmorphism, Grid, Flexbox, 애니메이션
- **JavaScript (Vanilla)** - DOM 조작, 이벤트 처리, 데이터 파싱
- **Canvas API** - 실시간 차트 렌더링 (외부 라이브러리 없음)
- **SVG** - 벡터 그래픽 다이어그램
- **localStorage** - 테마 설정 저장

### 백엔드 & 자동화
- **Python 3** - 데이터 처리 및 분석
  - `csv` 모듈 - CSV 파일 파싱
  - `datetime` - 날짜 계산 및 비교
  - `collections` - 데이터 집계

### 데이터 포맷
- **CSV** - 구조화된 데이터 저장
- **TXT** - 텍스트 기반 일정/메모
- **JSON** - 설정 파일
- **Markdown** - 보고서 형식

### 개발 도구
- **Git** - 버전 관리
- **Claude Code** - AI 기반 개발
- **Claude API** - AI 통합

## 🚀 사용 방법

### 대시보드 접근
1. `index.html`을 브라우저에서 열기
2. 또는 웹 서버에 배포하여 접근

### 데이터 업데이트
1. CSV 파일 수정:
   - `01_보고서/매출데이터.csv` - 판매 데이터
   - `03_업무/업무목록.csv` - 업무 항목
   - `01_보고서/프로젝트현황.csv` - 프로젝트 진행

2. TXT 파일 수정:
   - `03_업무/주간일정.txt` - 주간 일정
   - `02_메모/회의록.txt` - 회의 기록

### 김비서 AI 어시스턴트 실행
```
/김비서
```
- 모든 데이터 자동 수집
- 오늘 업무, 일정, 프로젝트 현황 분석
- 회의용 브리핑 자료 생성

### 다크/라이트 모드 토글
```javascript
const toggleTheme = () => {
  const current = document.documentElement.getAttribute('data-theme');
  const theme = current === 'light' ? 'dark' : 'light';
  document.documentElement.setAttribute('data-theme', theme);
  localStorage.setItem('theme', theme);
};
```
